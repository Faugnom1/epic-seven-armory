About E7 Armory

Author

Michael Faugno

Created

2024

Purpose

Analyze and display stats for units in Epic Seven.

Provide an overlay for Twitch.tv to display units for PVP.

Technologies

Flask - Web framework
Python - Backend processing
JavaScript - Frontend scripting
HTML/CSS - Webpage structure and styling
SQLAlchemy - Database management
Tesseract.js/PyTesseract - OCR
OpenAI API - Image analysis
Epic7db API - Game data
Smilegate API - Additional Unit name and data
Electron - Cross-platform desktop application
